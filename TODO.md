## Possible future improvements

- **Import from CSV/JSON** — the bulk import already supports comma-separated lines;
  a proper CSV or JSON file picker could make onboarding easier for users with many
  accounts.
- **Color coding** — allow an optional color per mapping and highlight the account row
  background for faster visual scanning.
- **Auto-detect unmapped accounts** — the content script could collect all account IDs
  seen on the page and report which ones have no mapping, surfacing them in the popup.
